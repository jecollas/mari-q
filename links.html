Other Websites I've Made

Click the title for links


Myth Generators is a collection of plot generators that I made for the plot team at Myth LARP. I'm one of the assistant producers there, and I wanted to make something that would help generate plots for our ever-growing player base.


Volmoon is my senior thesis. It is a data-driven interactive historical horror fiction game about the disturbing events of Bedburg, Germany in the fall of 1589. It uses original writing, illustration, and computer code to build an atmosphere of dread and tell this story how I believe it should be told. Created with Clip Studio Paint, inklewriter, HTML5, CSS, and Jekyll.


Charendo's Book of Heroes is a random generator that I made in my sophomore year of college for a programming class project. The purpose of this generator is to create a new Player Character for the tabletop roleplaying game, Dungeons and Dragons. Features of this generator include:

  • An accurate stat portrayal that organizes a character's with the class selection in mind (example: a bard would have Charisma as their highest stat).
  • A complete listing of all deities, races, subraces, classes, and backgrounds in the D&D 5e Player's Handbook and all other official Wizard's of the Coast material.
  • Multiple gender options that allow for the random character to be trans, nonbinary, or intersex.

Ariel Joy is a random generator that I created for a friend of mine in an attempt to spread a little bit more love and positivity.


Marvelverse was made with my friends in mind. Art block is a common problem amongst them, so I wanted to build a series of random generators that would supply them with ideas about what to draw. Web design by myself, side logo and banner design by Katie Thornton. This site includes:

  • Four random generators for art and writing prompts.
  • Character reference pages for all major characters.
  • A gallery for art dedicated to characters in the server.
  • A gallery for art dedicated to characters in the D&D campaign currently run by members of the server.
