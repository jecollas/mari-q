Random D&D Character Generator: This is a random generator that I made in my sophomore year of college for a programming class project. The purpose of this generator is to create a new Player Character for the tabletop roleplaying game, Dungeons and Dragons.
Features of this Generator Include:

  • An accurate stat portrayal that factors in bonuses from race selection and organizes them with the class selection in mind
    • (If the character created is a Halfling Bard, that character would have a bonus +2 to Dexterity and would have Charisma as their highest stat)
  • A complete listing of all deities, races, subraces, classes, and backgrounds in the D&D 5e Player's Handbook
  • Nonbinary gender option
